# Open Source Audit Project – Mozilla Firefox

This project is part of the Open Source Software course. The objective of this project is to analyze an open-source software and understand its philosophy, technical structure, and real-world usage.

I have chosen **Mozilla Firefox** as the subject of this audit. Firefox is an open-source web browser developed by the Mozilla Foundation, known for its focus on privacy, transparency, and user freedom.

## Project Overview

This project includes:

- A detailed report covering the origin, license, and ethical aspects of Firefox
- Analysis of how Firefox operates in a Linux environment
- Study of the open-source ecosystem around Firefox
- Comparison between Firefox and proprietary alternatives
- Implementation of 5 shell scripts demonstrating basic Linux skills

## Shell Scripts Included

1. **System Identity Report**  
   Displays system information such as kernel version, user name, uptime, and date.

2. **FOSS Package Inspector**  
   Checks whether Firefox is installed and displays package-related information.

3. **Disk and Permission Auditor**  
   Analyzes system directories and displays their size and permissions.

4. **Log File Analyzer**  
   Reads log files and counts occurrences of specific keywords.

5. **Open Source Manifesto Generator**  
   Generates a simple text-based manifesto based on user input.

## Technologies Used

- Ubuntu Linux (WSL)
- Bash Shell Scripting
- Mozilla Firefox

## Conclusion

This project helped in understanding the importance of open-source software and how Linux-based systems operate. It also provided hands-on experience with shell scripting and system-level operations.
